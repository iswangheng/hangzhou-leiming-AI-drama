## 🎯 核心原则（v0.6.2重大修正）

### **权重分配（v0.6.2修正）**
- **整体权重**:
  - **视觉元素**: 70%（构图、文字、表情、道具）
  - **听觉元素**: 30%（台词、BGM、音效）

- **钩子点判断权重** ⭐ **v0.6.2重大修正**:
  - **视觉元素**（字幕、人物状态、构图）：40%
  - **情节未完成**（时间节点、事件未完）：30%
  - **情绪未释放**（紧张、期待）：20% ⭐ **从10%提升**
  - **BGM配合**（戛然而止/静音）：10% ⭐ **从辅助提升为独立权重**

### **高光点"3秒抓眼球"原则**
```
0-1秒：视觉冲击（强对比色彩、夸张表情、关键道具）
1-2秒：悬念抛出（文字提示、未知危险）
2-3秒：类型确认（剧名、场景特征）
```

### **钩子点制造方式**（v0.6.2重大修正）
```
❌ 错误：识别"查看更多视频"按钮（实际不存在）
❌ 错误：认为BGM只是辅助信号（BGM占10%判断权重）

✅ 正确（6种真实钩子制造方式）：
   1. 时间节点（明确未来时间 + 未完成事件）
   2. 未完成事件（指出未完成的冲突，想知道结果）
   3. 对话留白（只给开头，想知道后续）
   4. 悬念结束时刻（关键行动或事件的高潮时刻画面停住）
   5. 情节重大转折（反转、突变、意外、命运改变）
   6. 纯画面型（无字幕，人物动作定格 + 情绪未释放 + BGM戛然而止）⭐ **v0.6.2新增**

✅ BGM配合（重要钩子信号，占10%权重）⭐ **v0.6.2修正**：
   - 人物动作定格 + BGM戛然而止 → 强钩子点（置信度+1.5）
   - 时间节点字幕 + BGM静音 → 强钩子点（置信度+1.5）
   - 未完成事件 + BGM静音 → 强钩子点（置信度+1.5）
   - 纯画面型（无字幕）+ 人物定格 + BGM静音 → 钩子点（置信度+0.5至+1.0）
   - 只有BGM静音 + 平淡场景 → 可能不是钩子点（需要谨慎判断）
```

## 🎨 视觉构图分析（v0.6新增）

### **镜头角度**
- **俯拍**: 第一视角代入感，常见于现代题材
- **仰拍**: 人物气势强化，常见于仙侠题材
- **主观视角**: 强烈代入感，常见于危险场景
- **平视**: 日常视角，生活化场景

### **景别类型**
- **特写**: 关键道具/表情强调，高光点标配
- **浅景深虚化**: 焦点集中，突出核心元素
- **中近景**: 展示人物状态，钩子点标配
- **全景**: 环境展示，场景信息

### **构图位置**
- **画面中心**: 核心元素，视觉焦点
- **黄金分割点**: 次要元素，平衡构图
- **背景虚化**: 突出前景，制造层次

### **色彩运用**
- **暖色调(黄/橙)**: 温馨、日常、现代题材
- **冷色调(蓝/绿)**: 神秘、危险、仙侠题材
- **对比色(冷暖)**: 冲突、奇幻、反转

## 📝 文字元素识别（v0.6新增）

### **文字位置**
- **画面中央**: 预警性文字（"小心"、"注意"）
- **画面底部**: 标题性文字（剧名、类型确认）
- **流动字幕**: 疑问性文字（好奇心缺口）
- **人物胸部**: 核心字幕（时间节点、未完成事件）

### **文字类型**
- **预警性**: 危险、冲突前奏
- **疑问性**: 好奇心缺口
- **时间节点**: 明确未来时间（"婚礼三天后举行"）
- **未完成事件**: 指出未完成的冲突

## 🎵 BGM信号分析（v0.6.2重大修正）

### **BGM是重要钩子信号，占10%判断权重** ⭐ **v0.6.2核心修正**

**为什么这样修正？**
- 人工剪辑台词质量低（转录多为乱码）
- 人工剪辑明确使用"结尾BGM戛然而止/静音"制造悬念
- 10%的钩子点没有字幕，完全依靠BGM + 情绪未释放

#### **强烈钩子信号**（需要其他线索配合）
- **BGM戛然而止** + 人物动作维持/定格 → 强钩子点（置信度 +1.5）
  - 示例：通话中人物 + BGM戛然而止
- **BGM静音** + 关键道具特写/表情定格 → 强钩子点（置信度 +1.5）
  - 示例：诊断报告特写 + BGM静音
- **BGM静音** + 时间节点字幕 → 强钩子点（置信度 +1.5）
  - 示例："婚礼三天后" + BGM静音
- **BGM静音** + 未完成事件字幕 → 强钩子点（置信度 +1.5）
  - 示例："还没给" + BGM静音

#### **中等钩子信号**
- **BGM戛然而止** + 有情节暗示 → 中等钩子点（置信度 +1.0）
  - 示例：追逐场景 + BGM戛然而止

#### **弱钩子信号**（需要谨慎验证）
- **只有BGM静音/戛然而止**，无明显视觉特征
  - 需要结合：人物动作、表情、场景
  - 如果：人物动作定格/微妙表情 + BGM静音 → 可能是钩子点（置信度 +0.5至+1.0）
  - 如果：平淡场景 + BGM静音 → 可能不是钩子点（置信度 -0.5）

#### **明确不是钩子点**
- **BGM维持正常** + 对话自然结束 → 不是钩子点（置信度 -1.0）
- **场景切换导致的停顿** + BGM静音 → 不是钩子点（置信度 -1.0）

## 🎯 高光类型

**描述**: 每部剧第1集开头的默认高光点

**视觉特征**:

- 特写、中景、近景
- ⭐ **v0.6**: 俯拍视角（代入感）、浅景深聚焦关键人物
- 说话、道歉、奔跑、下楼、踹人、倒地、发送短信
- 室内、谈判场所、楼梯、房间门口、婚礼现场、马路
**听觉特征**:
- 解释、道歉、愧疚、恳求、旁白、叙述、无奈、恐惧、自嘲、悲伤、绝望
- 揭露误会的真相、揭示人物关系、剧情突然转折、婚礼车祸
**典型场景**:
- 误会真相被揭露
- 人物关系发生逆转
- 婚礼当天发生意外
- 剧情出现重大转折

### 3. 信息揭示
**描述**: 通过信息揭示、身世揭秘、情感关系揭露等方式，快速抛出关键信息或揭示重要秘密，引发观众好奇心。
**视觉特征**:
- 平静、严肃、微笑、亲昵、惊讶、疑惑、心疼
- 特写、中景、近景
- ⭐ **v0.6**: 关键道具前置（文件、信件、照片）+ 浅景深虚化
- 低头、调整耳麦、整理头发、亲吻、站着、红眼眶
- 车内、演播厅、综艺节目现场、家庭饭桌、客厅
**听觉特征**:
- 旁白、叙述、调侃、疑问、安慰
- 揭示身世秘密、情感关系、人物背景
**典型场景**:
- 身世秘密被揭开
- 情感关系快速交代
- 关键信息抛出

### 4. 冲突爆发
**描述**: 剧情冲突爆发，通常伴随激烈的言语或肢体冲突，以及人物情绪的强烈变化。
**视觉特征**:
- 愤怒、惊讶、指责、隐忍、克制、质问
- 特写（人物面部、牌位）、中景、全景
- ⭐ **v0.6**: 推镜头（普通→特写）、人物占据画面中心
- 奔跑、指认、对峙、冲进、站立、低头

**听觉特征**:
- 质问、指责、指控、辩解、争吵
- 愤怒、激动、紧张、委屈、抱怨、无奈

**典型场景**:
- 婚礼现场发生冲突
- 会议现场激烈对峙
- 家庭矛盾公开爆发
