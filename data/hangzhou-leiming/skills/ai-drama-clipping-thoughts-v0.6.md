## 🎯 核心原则（v0.6更新）

### **权重分配**
- **视觉元素**: 70%（构图、文字、表情、道具）
- **听觉元素**: 30%（台词、BGM、音效）

### **高光点"3秒抓眼球"原则**
```
0-1秒：视觉冲击（强对比色彩、夸张表情、关键道具）
1-2秒：悬念抛出（文字提示、未知危险）
2-3秒：类型确认（剧名、场景特征）
```

### **钩子点制造方式**（v0.6修正）
```
❌ 错误：识别"查看更多视频"按钮（实际不存在）
✅ 正确：
   1. 剧情悬念（未完成情节、时间节点）
   2. 对话留白（只给开头、想知道后续）
   3. 情绪未释放（紧张、期待未缓解）
   4. 核心字幕（时间节点、未完成事件）
   5. BGM配合（结尾静音、戛然而止）
```

## 🎨 视觉构图分析（v0.6新增）

### **镜头角度**
- **俯拍**: 第一视角代入感，常见于现代题材
- **仰拍**: 人物气势强化，常见于仙侠题材
- **主观视角**: 强烈代入感，常见于危险场景
- **平视**: 日常视角，生活化场景

### **景别类型**
- **特写**: 关键道具/表情强调，高光点标配
- **浅景深虚化**: 焦点集中，突出核心元素
- **中近景**: 展示人物状态，钩子点标配
- **全景**: 环境展示，场景信息

### **构图位置**
- **画面中心**: 核心元素，视觉焦点
- **黄金分割点**: 次要元素，平衡构图
- **背景虚化**: 突出前景，制造层次

### **色彩运用**
- **暖色调(黄/橙)**: 温馨、日常、现代题材
- **冷色调(蓝/绿)**: 神秘、危险、仙侠题材
- **对比色(冷暖)**: 冲突、奇幻、反转

## 📝 文字元素识别（v0.6新增）

### **文字位置**
- **画面中央**: 预警性文字（"小心"、"注意"）
- **画面底部**: 标题性文字（剧名、类型确认）
- **流动字幕**: 疑问性文字（好奇心缺口）
- **人物胸部**: 核心字幕（时间节点、未完成事件）

### **文字类型**
- **预警性**: 危险、冲突前奏
- **疑问性**: 好奇心缺口
- **时间节点**: 明确未来时间（"婚礼三天后举行"）
- **未完成事件**: 指出未完成的冲突

## 🎵 BGM信号分析（v0.6新增）

### **结尾钩子BGM模式**
- **BGM静音**: 强烈钩子信号（置信度 +2.0）
- **BGM戛然而止**: 情绪中断，制造悬念
- **BGM高潮后骤降**: 预示重大转折
- **BGM维持正常**: 可能是假钩子（置信度 -1.5）

## 🎯 高光类型

### 1. 开篇高光
**描述**: 每部剧第1集开头的默认高光点
**视觉特征**:
- 开场画面、快速切入
- ⭐ **v0.6**: 特写强调、浅景深虚化突出核心元素
**听觉特征**:
- 开场音乐、快速对白
**典型场景**:
- 第1集开头

### 2. 剧情反转
**描述**: 剧情出现意想不到的转折，或人物关系、处境发生变化，引发观众好奇心。
**视觉特征**:
- 严肃、愧疚、惊讶、微笑、冷漠、痛苦、绝望
- 特写、中景、近景
- ⭐ **v0.6**: 俯拍视角（代入感）、浅景深聚焦关键人物
- 说话、道歉、奔跑、下楼、踹人、倒地、发送短信
- 室内、谈判场所、楼梯、房间门口、婚礼现场、马路
**听觉特征**:
- 解释、道歉、愧疚、恳求、旁白、叙述、无奈、恐惧、自嘲、悲伤、绝望
- 揭露误会的真相、揭示人物关系、剧情突然转折、婚礼车祸
**典型场景**:
- 误会真相被揭露
- 人物关系发生逆转
- 婚礼当天发生意外
- 剧情出现重大转折

### 3. 信息揭示
**描述**: 通过信息揭示、身世揭秘、情感关系揭露等方式，快速抛出关键信息或揭示重要秘密，引发观众好奇心。
**视觉特征**:
- 平静、严肃、微笑、亲昵、惊讶、疑惑、心疼
- 特写、中景、近景
- ⭐ **v0.6**: 关键道具前置（文件、信件、照片）+ 浅景深虚化
- 低头、调整耳麦、整理头发、亲吻、站着、红眼眶
- 车内、演播厅、综艺节目现场、家庭饭桌、客厅
**听觉特征**:
- 旁白、叙述、调侃、疑问、安慰
- 揭示身世秘密、情感关系、人物背景
**典型场景**:
- 身世秘密被揭开
- 情感关系快速交代
- 关键信息抛出

### 4. 冲突爆发
**描述**: 剧情冲突爆发，通常伴随激烈的言语或肢体冲突，以及人物情绪的强烈变化。
**视觉特征**:
- 愤怒、惊讶、指责、隐忍、克制、质问
- 特写（人物面部、牌位）、中景、全景
- ⭐ **v0.6**: 推镜头（普通→特写）、人物占据画面中心
- 奔跑、指认、对峙、冲进、站立、低头
- 宴会厅、会议现场、办公室、庭院、祠堂
**听觉特征**:
- 质问、指责、指控、辩解、争吵
- 愤怒、激动、紧张、委屈、抱怨、无奈
- 揭露关系、引发冲突
**典型场景**:
- 婚礼现场发生冲突
- 会议现场激烈对峙
- 家庭矛盾公开爆发

## 🎭 题材适配规则（v0.6新增）

### **现代题材特征**
- **场景**: 室内、办公室、医院、都市街道
- **道具**: 文件、手机、戒指、诊断报告、合同
- **色调**: 暖色调、生活化
- **钩子类型**: 时间节点、情感纠葛、现实冲突
- **镜头**: 俯拍（代入感）、浅景深
- **文字**: 预警性文字少、时间节点多

### **仙侠题材特征**
- **场景**: 夜晚、古宅、秘境、自然风光
- **道具**: 法宝、武器、古籍、符咒
- **色调**: 冷暖对比、神秘感
- **钩子类型**: 未完成事件、危险预警、法术对抗
- **镜头**: 仰拍（气势）、全景（环境）
- **文字**: 预警性文字多（"小心"）

### **重生题材特征**
- **场景**: 现代场景为主
- **道具**: 时间相关物品（日历、钟表）
- **色调**: 对比色调（前后对比）
- **钩子类型**: 时间重启、预知优势、命运改变
- **镜头**: 对比剪辑（前世vs今生）
- **文字**: 时间节点多

## ✅ 使用建议（v0.6）

### **AI分析提示**
1. **视觉优先**: 分析画面构图、镜头语言、色彩运用
2. **文字识别**: 识别字幕位置和类型
3. **BGM信号**: 结尾处BGM静音是强钩子信号
4. **题材适配**: 根据题材类型调整识别策略

### **质量过滤**
- 高光点置信度阈值：7.0
- 钩子点置信度阈值：7.0
- BGM静音时刻的钩子点：置信度 +2.0
- 视觉构图清晰的标记：置信度 +1.0
