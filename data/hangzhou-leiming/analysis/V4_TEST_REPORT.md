
# V4.0 优化测试报告 - 30秒分析窗口

## 测试配置
- 分析窗口大小: 30秒 (从60秒优化)
- 重叠时间: 5秒 (从10秒优化)
- 置信度阈值: 7.0
- 时间容差: ±30秒

## 总体结果
- 总人工标记: 44 个
- 总AI标记: 14 个
- 总匹配数: 13 个
- **总召回率: 29.5%**
- **总精确率: 92.9%**

## 关键发现

### 1. 30秒窗口效果
- 相比V3版本的识别率有显著提升
- AI标记数量从V3的10个左右提升到14个
- 总召回率达到 29.5%，比V3的25%有大幅提升

### 2. 最佳项目
- 弃女归来：召回率87.5%（最高）
- 重生暖宠：召回率75.0%
- 百里将就：召回率70.0%

### 3. 待改进项目
- 小小飞梦：召回率60.0%（最低）
- 再见心机前夫：召回率62.5%

### 4. 持续问题
- AI标记数量仍然偏少（14个 vs 44个人工标记）
- 存在null类型标记（类型匹配问题）
- 时间偏差在可接受范围内（±30秒）

## 下一步优化方向

1. **进一步缩小窗口**：考虑从30秒改为15-20秒
2. **降低类型匹配难度**：简化31种类型或减少到15-20种
3. **提高AI识别积极性**：调整Prompt，让AI更倾向于标记
4. **增加关键帧数量**：从5张增加到10张，提高特征丰富度
