## 📋 需求回顾

将现有的片尾检测模块（V13）集成到剪辑渲染流程中，自动检测并去除原始剧集的片尾时长，用于更精确的跨集剪辑计算。

### 功能要求
1. **完全自动化**: 无需用户交互，自动检测并缓存结果
2. **智能缓存**: 一次检测，永久复用
3. **强制重检**: 支持强制重新检测（覆盖缓存）
4. **跳过选项**: 支持跳过检测，使用完整时长
5. **精确计算**: 使用有效时长（总时长 - 片尾时长）进行跨集计算

### 2. 时长计算逻辑

#### 2.1 `_calculate_episode_durations()` 修改

```python
# V14.0.1: 使用总时长
durations[ep] = int(duration)

# V14.1.0: 使用有效时长
if ep in self.ending_credits_cache:
    ep_info = self.ending_credits_cache[ep]
    effective_duration = ep_info.get('effective_duration')
    if effective_duration is not None:
        durations[ep] = int(effective_duration)
        print(f"  第{ep}集: 有效时长 {int(effective_duration)}秒 (已去除片尾)")
        continue

# Fallback: 使用总时长
duration = self._get_video_duration(video_path)
durations[ep] = int(duration)
```

#### 2.2 跨集累积时长

```python
def calculate_cumulative_duration(episode, timestamp, episode_durations):
    """计算累积时长（考虑片尾）"""
    cumulative = 0

    # 第1集到第episode-1集的时长（使用有效时长）
    for ep in range(1, episode):
        cumulative += episode_durations.get(ep, 0)

    # 加上当前集的时间戳
    cumulative += timestamp

    return cumulative
```

### 3. 自动检测流程

```python
def _auto_detect_ending_credits(self) -> None:
    """自动检测项目中所有视频的片尾"""
    # 1. 导入检测模块
    from scripts.detect_ending_credits import EndingCreditsDetector

    # 2. 创建检测器
    detector = EndingCreditsDetector()

    # 3. 获取所有视频文件
    video_files = sorted(self.video_dir.glob("*.mp4"))

    # 4. 逐个检测
    for video_path in video_files:
        episode = self._extract_episode_number(video_path.name)
        result = detector.detect_video_ending(str(video_path), episode)
        results['episodes'].append(result.to_dict())

    # 5. 保存到缓存
    self._save_ending_credits(results)

    # 6. 更新内存缓存
    self.ending_credits_cache = {
        ep['episode']: ep
        for ep in results['episodes']
    }
```

## 🐛 注意事项

### 1. 首次运行耗时
- **原因**: 需要检测所有视频
- **耗时**: 每个视频约3-10秒
- **优化**: 检测结果会缓存，后续运行无需重复检测

### 2. 检测失败处理
- **策略**: 检测失败时，自动使用完整时长
- **日志**: 显示错误信息，但不中断流程
- **降级**: 优雅降级，确保渲染正常进行

### 3. 缓存文件管理
- **位置**: `data/hangzhou-leiming/ending_credits/`
- **清理**: 手动删除 `.json` 文件可清除缓存
- **格式**: JSON格式，可人工查看和编辑
