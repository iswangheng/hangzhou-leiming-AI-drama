### V15 - 视频花字叠加功能 (2026-03-05) 🎨

**核心功能**：
- ✅ **自动花字叠加**：在渲染剪辑时自动添加三行文本（热门短剧、剧名、免责声明）
- ✅ **10种预制样式**：金色豪华、红色激情、蓝色冷艳、紫色神秘、绿色清新、橙色活力、粉色浪漫、银色优雅、青色科技、复古棕色
- ✅ **项目级样式统一**：同一项目内所有剪辑使用相同样式（基于项目名称缓存）
- ✅ **左右交替显示**：热门短剧在左上/右上角交替显示，50%播放时间
- ✅ **朴素清晰设计**：剧名白色/淡紫色随机，细描边营造镂空效果
- ✅ **中文字体支持**：自动检测系统字体，优先使用Songti（宋体）

**使用示例**：
```bash
# 启用花字叠加（默认）
python -m scripts.understand.render_clips \
    data/hangzhou-leiming/analysis/项目名 \
    漫剧素材/项目名 \
    --add-overlay

# 不添加花字
python -m scripts.understand.render_clips \
    data/hangzhou-leiming/analysis/项目名 \
    漫剧素材/项目名 \
    --no-overlay
```

**花字内容**：
1. **热门短剧**（24号字体）
   - 位置：左上/右上角交替显示（每10秒切换）
   - 显示时长：50%播放时间（10秒显示+10秒隐藏循环）
   - 特效：倾斜-15度，金黄色字体，橙色描边

2. **《剧名》**（18号字体）
   - 位置：底部居中（距离底部90像素）
   - 显示时长：全程显示
   - 样式：白色或淡紫色随机，细描边（1.0）营造镂空效果

3. **免责声明**（12号字体）
   - 位置：底部居中（距离底部50像素，剧名下方40像素）
   - 显示时长：全程显示
   - 文案：随机选择（4种预设文案）

**技术特性**：
- FFmpeg drawtext滤镜实现
- 4层滤镜（热门短剧左+右，剧名，免责声明）
- 支持自定义字体、颜色、描边、阴影
- 自动避免遮挡原字幕（底部安全区）
- 样式缓存机制（项目级统一）

## 核心概念

### 高光点 ⭐
- **定义**：从**这个时刻开始**，观众更愿意看下去
- **特征**：吸引力最强的开始时刻（冲突爆发、悬念引入、情感高潮）
- **作用**：作为剪辑片段的**开始点**

### 钩子点 🪝
- **定义**：播放到**这个时刻突然结束**，观众迫切想知道后续
- **特征**：悬念最强的结束时刻（欲知后事如何、关键信息被截断）
- **作用**：作为剪辑片段的**结束点**

### 剪辑组合 🔗
```
高光点(开始) ───────────→ 钩子点(结束)
   └─ 吸引人进来           └─ 让人想继续
```
