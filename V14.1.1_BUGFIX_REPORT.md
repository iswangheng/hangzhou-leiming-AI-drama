## 📋 问题回顾

### 发现的问题

在首次运行V14.1完整渲染流程时，发现了两个严重问题：

#### 问题1: JSON序列化错误 ❌
**现象**: 片尾检测运行成功，但保存缓存时报错
```
❌ 片尾检测失败: Object of type bool is not JSON serializable
```

**根本原因**:
- `scripts/detect_ending_credits.py` 的 `EndingInfo.to_dict()` 方法返回的字典中包含 `numpy.bool_` 类型
- Python 标准库的 `json.dump()` 不能序列化 numpy 类型
- 导致缓存保存失败，`self.ending_credits_cache` 未更新
- 后续渲染无法使用有效时长，fallback到总时长

#### 问题2: 缺少结尾视频拼接参数 ❌
**现象**: 生成的视频文件没有 `_带结尾` 后缀

**根本原因**:
- 命令行未传递 `--add-ending` 参数
- `self.add_ending_clip = False`（默认值）
- 结尾视频拼接代码未执行

## ✅ 修复方案

### 修复1: numpy类型JSON序列化

**文件**: `scripts/detect_ending_credits.py`

**修改位置**: `EndingInfo.to_dict()` 方法（第60-78行）

**修改前**:
```python
def to_dict(self) -> dict:
    """转换为字典，确保所有字段都能被JSON序列化"""
    result = {
        'has_ending': self.has_ending,  # ❌ 可能是 numpy.bool_
        'duration': self.duration,
        'confidence': self.confidence,
        'method': self.method,
        'features': {}
    }

    # 手动处理features字段，确保所有值都能被JSON序列化
    for key, value in self.features.items():
        if isinstance(value, (str, int, float, bool, list, dict, type(None))):
            result['features'][key] = value
        else:
            result['features'][key] = str(value)

    return result
```

**修改后**:
```python
def to_dict(self) -> dict:
    """转换为字典，确保所有字段都能被JSON序列化

    V14.1修复: 强制转换numpy类型为Python原生类型
    """
    result = {
        'has_ending': bool(self.has_ending),  # ✅ 强制转换为Python bool
        'duration': float(self.duration),    # ✅ 强制转换为Python float
        'confidence': float(self.confidence), # ✅ 强制转换为Python float
        'method': str(self.method),
        'features': {}
    }

    # 手动处理features字段，确保所有值都能被JSON序列化
    for key, value in self.features.items():
        # 处理numpy类型和其他不可序列化的类型
        try:
            # 尝试JSON序列化测试
            import json
            json.dumps(value)

            # 如果成功，检查是否需要类型转换
            if hasattr(value, 'dtype'):  # numpy类型
                if value.dtype == 'bool':
                    result['features'][key] = bool(value)
                elif value.dtype in ['int64', 'int32']:
                    result['features'][key] = int(value)
                elif value.dtype in ['float64', 'float32']:
                    result['features'][key] = float(value)
                else:
                    result['features'][key] = value.item()  # numpy标量转Python类型
            elif isinstance(value, (str, int, float, bool, list, dict, type(None))):
                result['features'][key] = value
            else:
                result['features'][key] = str(value)
        except (TypeError, ValueError):
            # 如果序列化失败，转换为字符串
            result['features'][key] = str(value)

    return result
```

**关键改进**:
1. ✅ 强制转换 `has_ending` 为 `bool()`
2. ✅ 强制转换 `duration` 和 `confidence` 为 `float()`
3. ✅ 强制转换 `method` 为 `str()`
4. ✅ 检测并转换 `numpy.dtype` 类型
5. ✅ 使用 `value.item()` 转换 numpy 标量
6. ✅ 异常处理确保所有值都能序列化

## 🎯 技术细节

### numpy类型检测

**问题类型**:
- `numpy.bool_` - 不能被 `json.dump()` 序列化
- `numpy.int64`, `numpy.int32` - 需要转换
- `numpy.float64`, `numpy.float32` - 需要转换

**检测方法**:
```python
if hasattr(value, 'dtype'):  # 检测是否是numpy类型
    if value.dtype == 'bool':
        result['features'][key] = bool(value)
    elif value.dtype in ['int64', 'int32']:
        result['features'][key] = int(value)
    elif value.dtype in ['float64', 'float32']:
        result['features'][key] = float(value)
    else:
        result['features'][key] = value.item()  # 通用转换
```

### JSON序列化测试

**预防性测试**:
```python
try:
    import json
    json.dumps(value)  # 尝试序列化
    # 成功则使用原值
except (TypeError, ValueError):
    # 失败则转换为字符串
    result['features'][key] = str(value)
```
