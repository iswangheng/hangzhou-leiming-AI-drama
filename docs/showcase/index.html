剪辑流程

针对新短剧的智能剪辑完整流程

1. 上传新短剧

将新的短剧视频文件上传到OpenClaw平台

2. AI视频理解

系统自动提取关键帧、语音识别、分析视频内容

3. 识别剪辑点

基于训练的技能库，AI识别所有高光点和钩子点

4. 笛卡尔积组合

高光点 × 钩子点自动组合，生成所有可能的剪辑片段

5. 质量筛选

置信度>7.0 + 去重(10秒窗口) + 类型多样性 + Top 20智能排序

6. 敏感词检测

OCR+ASR双重检测，智能识别敏感词并自动马赛克遮盖

7. 花字叠加

自动添加热门短剧角标、剧名标题、免责声明

8. 片尾拼接

自动检测片尾位置，拼接片尾引导视频

9. 智能渲染

FFmpeg跨集单次编码 + 多线程并行渲染
