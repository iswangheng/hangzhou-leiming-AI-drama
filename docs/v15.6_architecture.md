# V15.6 代码结构说明

## 📁 模块架构

### 1. video_overlay.py - 花字叠加主模块（V15.6）

**职责**：
- 整合三个文字元素的叠加（热门短剧 + 剧名 + 免责声明）
- 管理样式系统（10种预设样式）
- 处理项目级样式缓存
- 构建FFmpeg滤镜链

**关键类**：
- `OverlayConfig`: 花字叠加配置（新增`hot_drama_position`参数）
- `VideoOverlayRenderer`: 视频渲染器

**技术实现**：
- **热门短剧**: 调用tilted_label.py生成PNG，使用overlay滤镜叠加
- **剧名**: 使用drawtext滤镜，动态字体大小（基于V2.3 v3算法）
- **免责声明**: 使用drawtext滤镜，动态字体大小（基于V2.3 v3算法）

**关键方法**：
- `apply_overlay()`: 主方法，生成倾斜角标PNG并构建滤镜链
- `_build_drawtext_filter()`: 构建drawtext滤镜（用于剧名、免责声明）

**V15.6关键改进**：
- ✅ **缩放逻辑统一**: 完全复制tilted_label的apply_label缩放逻辑，避免双重缩放
- ✅ **位置自定义支持**: 添加`hot_drama_position`参数，支持"top-left"和"top-right"
- ✅ **文件名优化**: 测试文件名包含分辨率和位置信息（如：锦庭别后意_360p_topleft_overlay.mp4）

**已移除的冗余代码**（V15.4清理）：
- `_build_alternating_enable()`: 不再支持左右交替显示
- `hot_drama_y`变量: 热门短剧位置现在由tilted_label模块自动计算
- 热门短剧的drawtext相关代码: 改用tilted_label的PNG预渲染

### 2. tilted_label.py - 倾斜角标模块（V4.9）

**职责**：
- 生成45度倾斜的半透明条幅角标
- 支持不同分辨率的动态字体大小缩放
- PNG预渲染（性能优化）
- **支持左上角和右上角位置**

**关键类**：
- `TiltedLabelConfig`: 倾斜角标配置
- `TiltedLabelRenderer`: 倾斜角标渲染器

**技术特点**：
- 大画布(400x400) + 超长色带(w=400)避免八边形问题
- 精准坐标偏移实现角落留白
- 95%不透明度（更明显的红色条幅）
- 动态字体大小：基于`resolution_ratio = smaller_dimension / 360.0`

**V4.9关键修复**：
- ✅ **投影计算错误修复**: 使用`canvas_half=200px`而非`projection=141px`计算overlay位置
- ✅ **数学正确性**: overlay的(x,y)是左上角，画布中心在(200,200)，位置计算必须正确
- ✅ **真正靠近角落**: 360p不再"过于靠中间"，所有分辨率视觉比例一致

**FFmpeg滤镜链**：
```bash
# 1. 创建透明画布
color=c=black@0:s=400x400,format=rgba[canvas]

# 2. 绘制半透明色块
[canvas]drawbox=x=0:y=170:w=400:h=60:color=red@0.95:t=fill[bg]

# 3. 绘制文字
[bg]drawtext=text='热门短剧':fontcolor=white:fontsize=XX...[txt]

# 4. 旋转45度
[txt]rotate=45*PI/180:c=black@0:ow=400:oh=400[rotated]
```

**V4.9位置计算**（修复后）：
```python
# 正确的位置计算（使用canvas_half而非projection）
canvas_half = 200  # 画布半径
corner_offset = 70  # 缩放后的值

if position == "top-right":
    # 右上角：画布中心在(W-corner_offset, corner_offset)
    x = video_width - corner_offset - canvas_half  # 1080p: 1080-168-200=712
    y = corner_offset - canvas_half                # 168-200=-32
else:  # top-left
    # 左上角：画布中心在(corner_offset, corner_offset)
    x = corner_offset - canvas_half  # 360p: 56-200=-144
    y = corner_offset - canvas_half  # 56-200=-144
```

**位置验证**（360p缩放后corner_offset=56）：
- 左上角: `x=-144, y=-144` (画布中心在56, 56)，距离边缘56px ✅
- 右上角: `x=104, y=-144` (画布中心在304, 56)，距离右边缘360-304=56px ✅

## 🔄 数据流

```
输入视频
    ↓
┌─────────────────────────────────────────┐
│ video_overlay.py                         │
│ ┌─────────────────────────────────────┐  │
│ │ 1. 获取视频尺寸和分辨率              │  │
│ │ 2. 计算动态字体大小（V2.3 v3算法）   │  │
│ └─────────────────────────────────────┘  │
│ ↓                                        │
│ ┌─────────────────────────────────────┐  │
│ │ tilted_label.py (热门短剧)          │  │
│ │ 1. 生成倾斜角标PNG (V4.1)           │  │
│ │ 2. 计算overlay位置                 │  │
│ └─────────────────────────────────────┘  │
│ ↓                                        │
│ ┌─────────────────────────────────────┐  │
│ │ drawtext滤镜 (剧名、免责声明)       │  │
│ │ 1. 构建drawtext滤镜                │  │
│ │ 2. 组合滤镜链                      │  │
│ └─────────────────────────────────────┘  │
│ ↓                                        │
│ FFmpeg执行:                             │
│ overlay(PNG) + drawtext(剧名) + drawtext │
│ (免责声明)                               │
└─────────────────────────────────────────┘
    ↓
输出视频（带完整花字叠加）
```

## 🎨 字体大小计算

### V2.3 v3算法（video_overlay.py - 剧名、免责声明）

```python
# 基于较小边计算分辨率倍数
smaller_dimension = min(video_width, video_height)
resolution_ratio = smaller_dimension / 360.0

# 基准字幕大小18px
base_subtitle_size = int(18 * resolution_ratio)

# 精简系数
hot_drama_font_size = int(base_subtitle_size * 1.2)
drama_title_font_size = int(base_subtitle_size * 0.95)
disclaimer_font_size = int(base_subtitle_size * 0.85)
```

**示例**：
- 360p: 基准18px, 剧名17px, 免责15px
- 1080p: 基准54px, 剧名52px, 免责46px

### V4.1算法（tilted_label.py - 热门短剧倾斜角标）

```python
# 基于较小边计算分辨率倍数
smaller_dimension = min(video_width, video_height)
resolution_ratio = smaller_dimension / 360.0

# 基准字体大小28px
tilted_font_size = int(28 * resolution_ratio)
```

**示例**：
- 360p: 28px
- 1080p: 84px (正好3倍)

## 📝 配置参数说明

### video_overlay.py配置

```python
OverlayConfig(
    enabled=True,
    style_id=None,              # None=随机，或指定样式ID
    project_name="",           # 项目名称（用于样式缓存）
```

```python

    label_text="热门短剧",
```
