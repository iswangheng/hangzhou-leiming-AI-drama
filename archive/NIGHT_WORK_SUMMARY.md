### 2. 核心算法优化

**ASR分析逻辑**：
1. **非常短的ASR（<1秒）+ 短文本（<5字）** → 视为噪音，忽略
2. **有片尾旁白关键词 + 短ASR** → 确认为片尾
3. **有正常剧情关键词 + 较长ASR（>1.5秒）** → 判定为误判
4. **短ASR但无明确关键词** → 视为片尾（可能是片尾的一部分）

**修正逻辑**：
- 无语音 + 画面相似 → 真实片尾
- 有语音 + ASR分析认为是片尾 → 真实片尾
- 有语音 + ASR分析认为不是片尾 → 误判
