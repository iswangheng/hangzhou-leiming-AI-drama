3. **AI遗漏题材特征** → 人工剪辑针对不同题材有不同策略
4. **AI误判"查看更多"** → 实际视频中不存在这个元素

## 一、人工剪辑 vs AI检测：特征对比表

### 1.1 高光点（开头）对比

| 维度 | 人工剪辑实际做法 | AI当前检测逻辑 | 差距分析 |
|------|-----------------|---------------|----------|
| **核心载体** | 视觉元素 (70%) + 音频 (30%) | ASR台词 (70%) + 视觉 (30%) | ❌ **权重倒置** |
| **时间要求** | 3秒内必须抓住观众 | 无明确时间要求 | ❌ **缺少时间紧迫性** |
| **视觉公式** | 情绪人物(30%) + 关键道具(30%) + 悬念文字(40%) | 表情识别 + 道具检测 (无权重) | ❌ **缺少文字分析** |
| **镜头语言** | 特写强调、浅景深虚化、俯拍代入感 | 仅识别表情，不分析镜头 | ❌ **遗漏构图分析** |

| **色彩运用** | 暖色调(日常) vs 冷色调(危险) vs 对比色(奇幻) | 无色彩分析 | ❌ **遗漏色彩维度** |

- AI遗漏的视觉特征占比：**60%**

### 3.1 "假钩子"误报问题 (影响精确度 -40%)

| 误报类型 | AI误判逻辑 | 人工实际标准 | 改进方案 |

| 场景切换停顿 | 视为"钩子点" | 不标记，正常转场 | 识别场景切换模式 |

| 情绪正常表达 | 视为"情感高潮" | 不标记，强度不够 | 增加"情绪强度阈值" |

```python
# 新增：scripts/extract_bgm.py
import librosa

def extract_bgm_features(audio_path):
    """
    提取背景音乐特征

    Returns:
        {
            "silence_moments": [12.5, 36.6, ...],  # BGM静音时刻 = 强钩子信号
            "volume_changes": [(10.2, "low_to_high"), ...],
            "tempo_changes": [(25.3, "slow_to_fast")]
        }
    """
    y, sr = librosa.load(audio_path)

    # 检测静音时刻
    sfms = librosa.feature.sfms(y=y)
    silence_frames = np.where(sfms < 0.1)[0]
    silence_times = librosa.frames_to_time(silence_frames, sr=sr)

    # 检测音量变化
    rms = librosa.feature.rms(y=y)
    volume_changes = detect_volume_changes(rms)

    # 检测节奏变化
    tempo, beats = librosa.beat.beat_track(y=y, sr=sr)
    tempo_changes = detect_tempo_changes(beats)

    return {
        "silence_moments": silence_times.tolist(),
        "volume_changes": volume_changes,
        "tempo_changes": tempo_changes
    }

# 集成到检测流程
# scripts/understand/analyze_segment.py
bgm_features = extract_bgm_features(audio_path)
if bgm_features["silence_moments"]:
    # BGM静音时刻 = 强烈钩子点信号
    hook_candidates.extend(bgm_features["silence_moments"])
```

```python
def is_true_hook_with_context(hook_candidate, before, after):
    """基于上下文判断真假钩子"""
```

## 八、关键风险与缓解措施

### 风险1：BGM分析误判
**风险**：正常BGM静音被误判为钩子点

**缓解措施**：
```python
# 增加复合判断
def is_hook_with_bgm(bgm_silence, other_features):
    """BGM静音 + 其他特征综合判断"""
    if bgm_silence:
        # 必须配合其他钩子特征
        if (other_features["has_unresolved_plot"] or
            other_features["has_key_info_cutoff"] or
            other_features["has_unreleased_emotion"]):
            return True
    return False
```

### 风险2：题材识别错误
**风险**：题材判断错误导致规则应用错误

**缓解措施**：
```python
# 增加置信度阈值
def apply_genre_rules(genre_result, threshold=0.7):
    """只有高置信度的题材判断才应用特定规则"""
    if genre_result["confidence"] >= threshold:
        return GENRE_RULES[genre_result["genre"]]
    else:
        return DEFAULT_RULES  # 回退到默认规则
```

### 风险3：Token截断问题依然存在
**风险**：即使优化Prompt，Token限制仍可能导致信息丢失

**缓解措施**：
```python
# 分段分析 + 结果合并
def segment_merge_analysis(video):
    """将长视频分段分析，然后合并结果"""
    segments = split_video(video, chunk_size=30)  # 30秒一段
    results = []

    for segment in segments:
        result = analyze_segment(segment)
        results.append(result)

    # 合并时去除重复标记
    merged_result = merge_and_deduplicate(results)
    return merged_result
```
